About the Set Analyzer

Hey! Thanks for stopping in an checking out my Set Analyzer. This was created as a way for me (as a broke college student) to calculate the expected value of Magic: The Gathering Packs. Feel free to explore this little website I made and have fun! Best of luck on the pack cracking!

Collector Booster Values in the Works!

This tool was programmed in python and uses the Scryfall API database to gather card values. The value index is a useful metric that compares the expected value of the cards in a booster pack to the cost of that pack.
